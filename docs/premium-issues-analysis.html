Point critique dans LanguageContext

Dans LanguageContext.tsx:21 :

const [language, setLanguageState] = useState(getCurrentLanguage());

getCurrentLanguage() retourne 'en' initialement (ligne 33 de i18n.ts), car initializeLanguage() n'a pas encore ete appelee.

Probleme identifie : Le composant premium.tsx peut se rendre avant que initializeLanguage() ne charge la langue sauvegardee.

Fix structurel pour LanguageContext.tsx

// apps/mobile/contexts/LanguageContext.tsx

export const useLanguage = () => {
  const context = useContext(LanguageContext);

  // Lire version pour forcer re-render quand langue change
  const { version } = context;

  const t: TranslateFunction = (key, options) => {
    return i18n.t(key, options);
  };

  return {
    language: context.language,
    setLanguage: context.setLanguage,
    isLoading: context.isLoading,
    version, // Exposer pour dependance
    t
  };
};

3. Audit : Package Matching RevenueCat

Code actuel (plus.tsx:38-53)

  (pkg) =>
    pkg.identifier.includes('premium_plus') ||
    pkg.identifier.includes('plus') ||

Problemes identifies

# Probleme Impact
1 includes('plus') trop vague Peut matcher des packages non desires
2 Aucun logging Impossible de debugger
3 IDs RevenueCat par defaut ($rc_monthly) Ne contiennent ni "plus" ni "premium_plus"

Fix propose pour plus.tsx

// Configuration en haut du fichier
const COSMIC_PACKAGE_CONFIG = {
  packageIdentifiers: [
    'premium_plus_monthly',
    'cosmic_monthly',
    '$rc_monthly',
  ],
  productIdentifiers: [
    'com.astrodating.cosmic.monthly',
    'com.astrodating.premium_plus',
  ],
};

// Remplacer loadPackages
const loadPackages = async () => {
  try {
    const packages = await getOfferings();

    // DEBUG: Logger tous les packages
    if (__DEV__) {
      console.log('[Plus] All RevenueCat packages:');
      packages.forEach((pkg, i) => {
        console.log(`  [${i}] identifier: "${pkg.identifier}"`);
        console.log(`      product: "${pkg.product.identifier}"`);
        console.log(`      price: ${pkg.product.priceString}`);
      });
    }

    let plusPkg;

    // 1. Chercher par package identifier exact
    for (const id of COSMIC_PACKAGE_CONFIG.packageIdentifiers) {
      plusPkg = packages.find(pkg => pkg.identifier === id);
      if (plusPkg) {
        console.log('[Plus] Found by package ID:', id);
        break;
      }
    }

    // 2. Fallback: product identifier
    if (!plusPkg) {
      for (const id of COSMIC_PACKAGE_CONFIG.productIdentifiers) {
        plusPkg = packages.find(pkg => pkg.product.identifier === id);
        if (plusPkg) break;
      }
    }

    if (plusPkg) {
      setPremiumPlusPackage(plusPkg);
    } else {
      console.warn('[Plus] No Cosmic package found!');
    }
  } catch (error) {
    console.error('[Plus] Failed:', error);
  }
};

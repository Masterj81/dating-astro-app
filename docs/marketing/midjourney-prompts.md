## Tips for Best Results

### Consistency Parameters
Add these to every prompt for brand consistency:
- `--style raw` - More literal interpretation
- `--s 250` - Moderate stylization
- `--q 2` - Higher quality (optional, uses more GPU)

### Aspect Ratios
- `--ar 1:1` - Square (Instagram feed, Facebook)
- `--ar 4:5` - Portrait (Instagram feed optimal)
- `--ar 9:16` - Vertical (Stories, Reels, TikTok)
- `--ar 16:9` - Landscape (YouTube, Twitter)
- `--ar 205:78` - Facebook cover

### Color Consistency
Reference these hex codes in prompts when needed:
- Deep Space: #0f0f1a
- Cosmic Purple: #1a1a2e
- Stellar Pink: #E94560
- Gold Star: #FFD700

### Remix/Variations
After generating, use:
- `--seed [number]` to recreate similar images
- Vary (Strong) for different interpretations
- Vary (Subtle) for minor adjustments
- Pan/Zoom for extending images

### Upscaling
Always upscale final selections:
- Use `U1`, `U2`, `U3`, or `U4` buttons
- For social media: 2x upscale is usually enough
- For print: Use 4x or Upscale (Creative)

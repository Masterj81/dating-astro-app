## Assembly Instructions

### Step 1: Generate Images
1. Copy each Midjourney prompt
2. Paste in Midjourney Discord: `/imagine prompt: [paste here]`
3. Wait for generation
4. Pick best variation (V1-V4)
5. Upscale (U1-U4)
6. Download image

### Step 2: Add Text in Canva
1. Go to canva.com
2. Create design → Instagram Post (1080 x 1350 px)
3. Upload Midjourney image as background
4. Add text overlays from storyboard
5. Use fonts:
   - Headlines: Playfair Display or Bodoni
   - Body: Inter or Montserrat
6. Colors: White text with subtle drop shadow

### Step 3: Export
1. Download all slides as PNG
2. Name files: carousel-name-01.png, carousel-name-02.png, etc.
3. Upload to Instagram in order

### Step 4: Post
1. Select all slides in order
2. Add caption from storyboard
3. Add hashtags (first comment or end of caption)
4. Post at optimal time (11AM-1PM or 7PM-9PM)

## Pro Tips

### Consistency
- Use same background gradient across all slides
- Keep text in same position on each slide
- Use consistent icon style

### Engagement Hooks
- Slide 1 should create curiosity
- Last slide always has CTA
- Middle slides deliver value

### Saving/Sharing
- Educational content gets saved
- Controversial content gets shared
- Relatable content gets comments

### Text Readability
- Maximum 20-30 words per slide
- High contrast (white on dark)
- Large font size (minimum 24px)

## Production Notes

### Video Format
- Aspect ratio: 9:16 (vertical)
- Duration: 30 seconds
- Platform: Instagram Reels, TikTok, YouTube Shorts

### Midjourney Settings
- All prompts use `--ar 9:16` for vertical format
- `--style raw` for consistent photorealistic brand look
- `--s 250` for moderate stylization
- `--no text words letters` prevents Midjourney from adding unwanted text
- Use `--v 6.1` for best results (add to each prompt if needed)

### Key Improvements Over V1 Prompts
1. **Concrete visual descriptions** instead of abstract concepts ("energy", "visualization")
2. **Specific objects and compositions** that Midjourney renders well
3. **Sign-specific color palettes** (Aries=gold/fire, Cancer=silver/moon, etc.)
4. **Negative prompts** (`--no text words`) to prevent unwanted text generation
5. **Camera/composition language** (close-up, wide shot, overhead, shallow depth of field)
6. **Material descriptions** (obsidian, rose gold, translucent, crystalline) for better texture

### Post-Generation Workflow
1. Generate all 7 scenes per sign in Midjourney
2. Upscale selected images (U1-U4)
3. Import to video editor (CapCut, Premiere, After Effects)
4. Add text overlays matching the script
5. Add trending audio/music
6. Export at 1080x1920

The final CTA scene (28-30s) is identical for all signs — generate once and reuse.

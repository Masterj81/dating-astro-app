# AstroDating Instagram Reels & TikTok Storyboards

Complete storyboards with scene breakdowns, timing, text overlays, Midjourney prompts, and AI video prompts.

---

## How to Use This Guide

### Option A: Midjourney + CapCut (Easiest)
1. Generate static images with Midjourney prompts
2. Import to CapCut
3. Add Ken Burns effect (zoom/pan)
4. Add text overlays and transitions
5. Add trending audio

### Option B: AI Video Tools (Best Quality)
1. Use Runway ML, Pika, or Kling prompts
2. Generate video clips
3. Combine in CapCut
4. Add text and audio

### Option C: Hybrid
1. Generate key frames in Midjourney
2. Animate with Runway "Image to Video"
3. Combine in CapCut
